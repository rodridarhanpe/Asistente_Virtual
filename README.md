Este proyecto es un asistente virtual hecho en Python como práctica personal para aprender a trabajar con librerías externas, reconocimiento de voz y automatización de tareas simples.
El asistente puede escuchar comandos, responder con voz y ejecutar distintas acciones básicas según lo que el usuario le pida.

¿Qué puede hacer?

- Escuchar comandos por micrófono
- Responder utilizando voz
- Buscar información en Wikipedia
- Abrir páginas web
- Decir la hora y la fecha
- Contar chistes
- Consultar información financiera básica
- Ejecutar acciones simples según el comando recibido

El proyecto utiliza las siguientes bibliotecas de Python:

pyttsx3,speech_recognition,pywhatkit,yfinance,pyjokes,wikipedia,webbrowser,datetime

Cómo ejecutar el proyecto

Seguí estos pasos desde la terminal o desde PyCharm:

Clonar el repositorio
git clone https://github.com/tu-usuario/tu-repositorio.git

Entrar a la carpeta del proyecto
cd tu-repositorio

Crear y activar un entorno virtual

python -m venv venv
source venv/bin/activate

Instalar las dependencias
pip install -r requirements.txt

Ejecutar el asistente virtual
python asistente_virtual.py

Una vez iniciado, el asistente indicará cuándo podés comenzar a hablar.

El proyecto está pensado con fines educativos.
Puede presentar errores dependiendo del sistema operativo o del micrófono.
Se recomienda ejecutar el proyecto desde la terminal o PyCharm.
